Preparatory steps

You can install Ubuntu Touch on the versions F5121 and F5122 of the Sony Xperia X. Before, you have to take a number of preparatory steps:

Ensure you have upgraded the stock firmware at least to Android 8, else you can't flash the OEM binaries for AOSP. It is best to ensure that you have all the latest firmware installed.

Enable developer options for your device.

  • Actors:

  • + Manual brightness
  • + Notification LED
  • - Torchlight
  • - Vibration

  • Camera:

  • + Flashlight
  • - Photo
  • - Video

  • Cellular:

  • + Carrier info, signal strength
  • + Data connection
  • + Incoming, outgoing calls
  • - MMS in, out
  • - PIN unlock

  • Cellular:

  • + Change audio routings
  • + Voice in calls
  • - SMS in, out

  • Network:

  • + Bluetooth
  • + Flight mode
  • - Hotspot

  • Endurance:

  • + Battery lifetime > 24h from 100%
  • + No reboot needed for 1 week

  • GPU:

  • + Boot into UI
  • + Video acceleration

  • Misc:

  • + Anbox patches applied to kernel
  • + Battery percentage
  • + Offline charging
  • + Online charging
  • + Wirelss charging
  • + Recovery image
  • + Reset to factory defaults
  • + Shutdown / Reboot

  • Network:

  • + NFC
  • + WiFi
  • + Automatic brightness

  • Sensors:

  • + Fingerprint reader
  • + GPS
  • + Proximity
  • + Rotation
  • + Touchscreen

  • Sound:

  • + Earphones
  • + Loudspeaker
  • + Microphone
  • + Volume control

  • USB:

  • + MTP access
  • + RNDIS access
  • + External monitor
